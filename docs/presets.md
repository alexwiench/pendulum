# Presets

Presets let you save easing curves and recall them later with a single click. Each preset is shown as a small visual thumbnail so you can identify curves at a glance.

## Saving Presets

- **Save current curve**: Click the **+** button in the preset bar to save your active curve.
- **Save ghost curves**: When ghost curves are visible, a second save button appears. Click it to save all currently visible ghost curves as individual presets. This is useful for capturing existing easing before you modify it.

## Preset Thumbnails

Each saved preset displays as a miniature speed graph. The thumbnail shows the same shape you'd see on the full canvas, making it easy to visually browse your library.

## Loading a Preset

Click any preset thumbnail to load that curve into the editor. If auto-apply is enabled, it applies to your After Effects keyframes immediately.

## Deleting a Preset

Hover over a preset to reveal the delete button (a red X in the corner). Click it to remove that preset.

## Storage

- Presets persist between After Effects sessions (stored in the panel's local storage).
- Maximum **20 presets**. When the limit is reached, the oldest preset is automatically removed to make room.
- If you have many presets, the bar scrolls horizontally.
- Presets are local to your machine and are not included in settings export/import. To back them up, you would need to copy the panel's storage data.
